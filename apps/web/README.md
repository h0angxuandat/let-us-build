# @let-us-build/web

SolidStart web UI for let-us-build. **M0 = scaffold only** (not installed, not wired to the API).

Screens (dashboard, 3-step new-project wizard, requirement intake, 4-lane Kanban, ticket drawer
with the live **Discussion** tab, agent settings) land in **M2** per
[`../../design/ui-ux-spec.md`](../../design/ui-ux-spec.md).

## Dev (from M2 onward)
```bash
pnpm install
pnpm dev        # http://localhost:8301
```
